Dragonfruit

This game This game is made for an intership at SyncVR, a game with the target group children. This game is made to relax. Fly around as a dragon and catch all the fruit. No need to rush, there is no time limit.

Download APK and install on android. Make sure the phone is able to install .APK's not downloaded via de play or app store. Game isn't playable on PC since it uses the gyro in the phone to fly the character. Game is not tested on Iphone.
