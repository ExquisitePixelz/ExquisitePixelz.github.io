PeakUnbound

My passion project combines VR gaming with real life climbing. This game is a continuation from Hybrid Heist, where we as a team created a game for the quest 1

PeakUnbound is a game where players climb on real life bouldering routes and in VR see a wonderful cave system. The route the climbers climb is synced with the real life route. And using handtracking, climbers are able to navigate virtual spaces and experience true climbing. In this cave system, a puzzle needs to be solved, one you can only solve by climbing. Follow the narrative, explore the cave and climb above dizzying heights. Watch out for breaking grips as you try and hold on. This game is a passion project of mine where I combined both my hobbies into one and asked myself the question: 'Why not?'. With oustanding feedback and enthousiasme we took the game a step further and contacted several bouldering halls where we tested our game with several climbers during awesome events.

The game is an experiment to see if we could use this as an training program for climbing. We can manipulate the difficulty of climbing by removing or adding climbing grips inside the game.
